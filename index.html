About

About Study Night

Study Night is a small front-end project built to practice application setup, code quality, optimization, unit testing, end-to-end testing, and task automation.

The project uses Parcel for production builds, Gulp for task automation, Mocha and Chai for unit testing, and Cypress for browser workflow testing.
